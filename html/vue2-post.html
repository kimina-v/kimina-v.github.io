HTML 속성에 접근 v-bind
HTML 속성에 접근하려면, 디렉티브를 사용해야 한다.
디렉티브란? v- 접두사가 붙는것이 특징.
EX1
HTML
    
        <div id="app">
            <a v-bind:href="link">링크</a>
        </div>  
    
JS
    
        var app = new Vue({
            el:"#app", 
            data:{
                link:"http://kimina-v.github.io"
            },
            methods:{
            }
        })
methods 내에 howAreYou()라는 함수를 선언
결과

약어
자주 사용되는 v-bind와 v-on에 대해 특별한 약어 제공
v-bind 약어 :
HTML
    
        // 전체 문법
        <a v-bind:href="url">.......</a>

        // 약어 
        <a :href="url">.......</a>
v-on 약어 @
HTML
    
    // 전체 문법
    <a v-on:click="doSomething">.......</a>

    // 약어 
    <a @click="doSomething">.......</a>
일회성 디렉티브 v-once
EX1
HTML
    
        <div id="app2">
            <h1>{{ title }}</h1>
            <p>{{ sayHello() }}</p>
        </div>  
    
JS
    
        var app2 = new Vue({
            el:"#app2", 
            data:{
                title:"Hello kimina world!"
            },
            methods:{
                sayHello: function(){
                    this.title = "Have a good time"
                    return this.title
                }
            }
        })
methods 내에 howAreYou()라는 함수를 선언
결과

{{ title }}

{{ sayHello() }}

{{ title }}은 변경 되기 전 값이 출력되고, {{ sayHello() }}는 변경이 된 이후 값으로 출력되도록 할 떄 사용되는 것이 v-once디렉티브 이다.
HTML
    
        <div id="app3">
            <h1 v-once>{{ title }}</h1>
            <p>{{ sayHello() }}</p>
        </div>  
    
JS
    
        var app2 = new Vue({
            el:"#app3", 
            data:{
                title:"Hello kimina world!"
            },
            methods:{
                sayHello: function(){
                    this.title = "Have a good time"
                    return this.title
                }
            }
        })
결과

{{ title }}

{{ sayHello() }}

HTML코드를 직접 넣어주는 v-html
EX1
HTML
    
        <div id="app4">
            <p v-html="aLinkToMedium"></p>
        </div>  
    
JS
    
        var app2 = new Vue({
            el:"#app4", 
            data:{
                aLinkToMedium: " <a href='http://kimina-v.github.io' class='link'>kimina-v </a>"
            }
        })
결과
