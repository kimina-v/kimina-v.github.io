양방향 데이터 바인딩 v-model
EX1
JS
    
        var app = new Vue({
            el:"#app", 
            data:{
               name: "kimina"
            }
        })
    
HTML
    
        <div id="app">
            <input type="text" v-model="name">
            {{ name }}  
        </div>
결과
{{ name }}
조건부 렌더링 v-if
EX1
JS
    
        var app = new Vue({
            el:"#app2", 
            data:{
               dog: true
            }
        })
    
HTML
    
        <div id="app2">
            <p v-if="dog">강아지가 좋아 ❤</p>
            <p v-else>무슨소리 고양이가 좋아❤</p>
            <button v-on:click="dog = !god">반대</button>
        </div>
결과
강아지가 좋아 ❤ 무슨소리 고양이가 좋아❤
EX2
template 활용한 예제
HTML
    
        <div id="app3">
            <p v-if="dog">강아지가 좋아 ❤</p>
            <p v-else>무슨소리 고양이가 좋아❤</p>
            <button c-on:click="dog = !god">반대</button>
            <template v-if="dog">
                <p>강아지는 귀여워</p>
                <p>집에 오면 강아지는 날 반겨줘</p>
                <p>강아지는 가족이야</p>
            </template>
            <template v-else>
                <p>고양이도 귀여워</p>
                <p>고양이는 시크한 매력이 있어</p>
                <p>고양이도 가족이야</p>
            </template>
        </div>
결과
강아지가 좋아 ❤ 무슨소리 고양이가 좋아❤
v-show
v-if 디렉티브와 비슷한 역할을 한다.
EX1
JS
    
        var app = new Vue({
            el:"#app4", 
            data:{
               dog: false
            }
        })
    
HTML
    
        <div id="app4">
            <p v-if="dog">강아지가 좋아❤</p>
            <p v-show="dog">귀여운 댕댕이</p>  
        </div>
결과

강아지가 좋아❤

귀여운 댕댕이

결과 확인을 위해 개발자도구에서 확인.
v-if : 조건에 따라 아예 코드를 지운다.
v-show : 코드는 있지만, 스타일로 display:none처리한다.
리스트 렌더링 v-for
EX1
JS
    
        var app = new Vue({
            el:"#app5", 
            data:{
               list: ['사과','복숭아','딸기','오렌지','수박']
            }
        })
    
HTML
    
        <div id="app5">
            <h1>과일 종류<h1>  
            <ul>
                <li v-for="item in list"{{ item }}></li>
            </ul>
        </div>
결과

과일 리스트

  • {{ item }}
EX2
v-for내 현재 index 값을 알아보자
HTML
    
        <div id="app6">
            <h1>과일 종류<h1>  
            <ul>
                <li v-for="(item, i) in list">{{ item }} ({{ i }})</li>
            </ul>
        </div>
결과

과일 리스트

  • {{ item }} ({{ i }})
string이 아닌 object의 반복
EX1
JS
    
        var app = new Vue({
            el:"#app7", 
            data:{
                travelList:[
                    {name: '강릉', distance: '차로 3시간'},
                    {name: '부산', distance: '차로 5시간'},
                    {name: '춘천', distance: '차로 2시간'},
                    {name: '타이베이', distance: '비행기로 3시간'},
                    {name: '마닐라', distance: '비행기로 5시간'},
                    {name: '스페인', distance: '비행기로 17시간'},
                ]
            }
        })
    
HTML
    
        <div id="app7">
            <h1>여행지 리스트</h1>
            <ul>
                <li v-for="city in travelList">{{ city }}</li>
            </ul> 
        </div>
결과

여행지 리스트

  • {{ city }}
object가 그대로 렌더링 된다. 우리가 원하는 결과 값을 얻이 위해서는 city.속성으로 써주면 된다.
HTML
    
        <div id="app7">
            <h1>여행지 리스트</h1>
            <ul>
                <li v-for="city in travelList">
                    {{ city.name }} ~ {{ city.distance }}
                </li>
            </ul> 
        </div>
결과

여행지 리스트

  • {{ city.name }} ~ {{ city.distance }}
