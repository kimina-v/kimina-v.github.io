자동 계산된 속성 computed
EX1
버튼 클릭시 클릭한 횟수가 +1씩 증가하는 application을 만들어보자.
클릭한 횟수 저장 변수 count
JS
    
        var app = new Vue({
            el:"#app4", 
            data:{
               count: 0 
            },
            computed:{
                isLargerThanTen: function(){
                    return this.count * 3 > "10보다 큽니다." : "10보다 작거나 같습니다."
                }
            }
        })
    
HTML
    
        <div id="app4">
            {{ isLargerThanTen }}
            <button v-on:click="count ++">Click!</button>    
        </div>
결과
{{ isLargerThanTen }}
computed 대신 methods로 바꾸고, {{ isLargerThanTen }}를 {{ isLargerThanTen() }}로 바꾸면 methods를 활용해서 같은 결과 볼 수 있다.
computed VS methods
EX1
JS
    
        var app = new Vue({
            el:"#app5", 
            data:{
               count: 0 
            },
            computed:{
                double: function(){
                    return this.count * 2
                }
            },
            methods: {
                triple: function(){
                    return this.count * 3
                }
            }
        })
    
HTML
    
        <div id="app5">
            <span>count : {{ count }}</span>
            <span>computed: {{ double }}</span>
            <span>methods : {{ triple() }}</span>
            <button v-on:click="count ++">Click!</button>    
        </div>
결과
count : {{ count }} computed : {{ double }} methods : {{ triple() }}
EX2
methods 와 computed의 차이점을 찾아보자.
JS
    
        var app = new Vue({
            el:"#app6", 
            data:{
               count: 0,
               secondCount: 0
            },
            computed:{
                double: function(){
                    console.log('Computed double')
                    return this.count * 2
                }
            },
            methods: {
                triple: function(){
                    console.log('methods triple')
                    return this.count * 3
                }
            }
        })
    
HTML
    
        <div id="app6">
            <span>count : {{ count }}</span>
            <span>second count : {{ secondCount }}</span>
            <span>computed: {{ double }}</span>
            <span>methods : {{ triple() }}</span>
            <button v-on:click="count ++">Click!</button>    
            <button v-on:click="count ++">Click!</button>    
            <button v-on:click="secondCount ++">Click!</button>    
        </div>
결과
count : {{ count }} second count : {{ secondCount }} computed : {{ double }} methods : {{ triple() }}
