    이번 주제 – 🔸조인성이 직접 뽑은 맛있닭 TOP3🔸

    랭킹닭컴 이벤트 게시판에서 진행 중인 NEW 모델 ‘누구시죠?’
    이벤트 보신적 있으신가요? 다들 이미 알고 계시겠지만
    랭킹닭컴 올해의 모델은
    바로 믿.보.배 (믿고 보는 배우) 조인성씨입니다.

    오랫동안 팬심을 가져왔던 분들이 주위에 많아서
    조인성 씨가 랭킹닭컴 모델로 선정되었다는 소식이 돌자
    반응이 아주아주 뜨거웠는데요.
    그런 조인성씨가 모델로 선정되고 난 후,
    제가 가장 많이 받았던 질문이 뭐였는지 아세요?

    바로 “조인성 씨 실물은 더 잘생겼지?”였습니다.

    닭가슴살에 단백질이 많냐는 것만큼 당연한 질문이라
    어떻게 답변을 해야할지 애매했지만
    실제 조인성씨를 만났던 상황을 여러분께
    설명 드리자면 이렇습니다.

    “조인성 씨 언제와? 어 왔다, 헐!”… 웅성웅성👤👥👤👥..
    “너무 미친듯이 잘생겼다..” 인성웅성👤👥👤👥..
    “다리 길이가 제 가슴까지 오는 것 같아요..” 웅성웅성👤👥👤👥..
    “같이 사진 찍으면 오징어 되겠죠?”.. 웅성인성👤👥👤👥
