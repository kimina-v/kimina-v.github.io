vue.js
사용자 인터페이스를 만들기 위한 프로그레시브 프레임워크
다른 라이브러리나 기존 프로젝트와의 통합이 매우 쉽다.
vue.js 정보저장소 data
EX1
HTML
    
        <div id="app">
            {{ title }}
        </div>  
    
JS
    
        new Vue({
            el:"#app", // el :  Vue가 새롭게 정의될 때, 해당 Vue가 적용될 요소를 지정해주는 역할
            data:{ // data : 해당 VueJS에서 사용할 정보들을 선언해주는 역할
                title:"Hello kimina world!"
            }
        })
결과
{{ title }}
vue.js 함수저장소 methods
EX1
HTML
    
        <div id="app2">
            <p>{{ title }}</p>
            <p>{{ howAreYou() }}</p>
        </div>  
    
JS
    
        var app2 = new Vue({
            el:"#app2", 
            data:{
                title:"Hello kimina world!"
            },
            methods:{
                howAreYou: function(){
                    retuen "기분 어때?"
                }
            }
        })
methods 내에 howAreYou()라는 함수를 선언
결과

{{ title }}

{{ howAreYou() }}

vue를 지칭하는 단어 this
EX1
HTML
    
        <div id="app3">
            <p>{{ sayHello() }}</p>
            <p>{{ howAreYou() }}</p>
        </div>  
    
JS
    
        var app2 = new Vue({
            el:"#app3", 
            data:{
                title:"Hello kimina world!"
            },
            methods:{
                howAreYou: function(){
                    retuen "기분 어때?"
                },
                sayHello: function(){
                    retuen this.title
                }
            }
        })
methods 내에 howAreYou()라는 함수를 선언
결과

{{ sayHello() }}

{{ howAreYou() }}
