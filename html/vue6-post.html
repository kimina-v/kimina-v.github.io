화살표 함수
EX1
JS
    
        new Vue({
            data: {
                size: "large", // 데이터 속성
                items: [{size:"small"},{size:"large"}]
            },
            computed: {
                filterBySize(){ //computed 속성
                    let size = this.size; // 콜백함수 안에 뷰 컴포넌트의 데이터 속성을 접근하기 위한 추가 코드
                    return this.items.filter(function(item){
                        return item.size === size;
                        // ! 여기서 this.size를 접근하면 undefined
                    })
                }
            }
        })

인자 값이 1개인 경우
화살표 함수는 특정 상황에서 더 간결하게 작성 가능하다.
함수 인자 값이 1개라면 ()생략가능
함수 정의 로직이 1줄이면 {}생략 가능

템플릿 리터럴
기존 문자열 표시할 떄 사용하는 작은 따옴표('), 큰따옴표(") 대신 백틱(`) 사용
문자열을 여러 줄에 걸쳐 표시 가능(뷰 컴포넌트의 템플릿 선언 시 유용)
문자열과 자바스크립트 표현식을 함꼐 사용하기 좋음(computed 속성 사용 편함)
